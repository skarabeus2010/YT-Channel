# Controller-Checkliste — <Video-Titel>

> Pro Video kopieren. Vom **Controller** (Agent 05) ausgefüllt. Belege (Link/Screenshot) je Befund.

## 6.1 Pre-Production
- [ ] Klare 3-Sek.-Hook + Retention-Struktur (Mehrwert in ersten 30–60 Sek.)
- [ ] Suchnachfrage + Lücke belegt
- [ ] Titel (Curiosity-Gap) + Keywords definiert

## 6.2 Historische Korrektheit ⚠ HARD-STOP
- [ ] Alle Sach-Aussagen mit Quelle im Quellen-Log belegt
- [ ] Fakt vs. dramatisierte Rekonstruktion klar getrennt
- [ ] Keine KI-Halluzination als Fakt; keine erfundene „Archiv"-Aufnahme als echt
- [ ] Visuals epochen-plausibel (kein Anachronismus)

## 6.3 KI-Compliance ⚠ HARD-STOP
- [ ] YouTube „synthetic/altered content"-Label gesetzt
- [ ] C2PA-Metadaten eingebettet
- [ ] Disclosure-Text in Description
- [ ] KI-Stimme offengelegt · EU-AI-Act-Kennzeichnung erfüllt

## 6.4 Anti-Slop / Qualität
- [ ] Eigener Winkel + recherchierte Tiefe (kein Massen-Template)
- [ ] Menschliche Redaktion dokumentiert; nicht 100 % synthetisch wo vermeidbar
- [ ] Audio/Visual sauber; Captions synchron; Musik Content-ID-sicher (Lizenz notiert)

## 6.5 SEO / Packaging
- [ ] Titel+Thumbnail = eine Curiosity-Einheit
- [ ] Keyword im 1. Description-Satz · Chapters (00:00, ≥3) · 3–5 Hashtags · Captions
- [ ] Thumbnail-A/B vorbereitet
