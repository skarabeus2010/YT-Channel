# Glossar

> Kurz erklärt — die wichtigsten Begriffe für dieses Projekt.

| Begriff | Bedeutung |
|---|---|
| **Faceless** | Kanal ohne sichtbare Person; Inhalt aus Voice-over + Visuals (hier KI-generiert). |
| **CPM** | *Cost per Mille* — was Werbetreibende pro 1.000 Anzeigen zahlen (Bruttowert). |
| **RPM** | *Revenue per Mille* — was **du** netto pro 1.000 Views verdienst (nach YouTube-Anteil). |
| **CTR** | *Click-Through-Rate* — Anteil der Impressionen, die zu Klicks werden. Größter Wachstums-Hebel. |
| **Retention / AVD** | Audience Retention = wie viel % im Schnitt geschaut wird; AVD = durchschnittliche Sehdauer. |
| **Impressionen** | Wie oft dein Thumbnail Nutzern gezeigt wurde. |
| **Returning Viewers** | Wiederkehrende Zuschauer; >10 % = Algorithmus hat ein Publikum gefunden. |
| **YPP** | *YouTube Partner Program* — Monetarisierung; Schwelle 1.000 Abos + 4.000 Std. Watchtime (oder 10 Mio. Shorts-Views/90 T.). |
| **Browse / Suggested / Search** | Traffic-Quellen: Startseite / „Nächstes Video" / Suche. Neue Kanäle leben oft von Search + Shorts. |
| **Packaging** | Titel + Thumbnail als Einheit — entscheidet über den Klick. |
| **Hook** | Die ersten ~3 Sekunden, die zum Weiterschauen bewegen. |
| **AI-Slop** | Abwertend: massenproduzierter, billiger KI-Content ohne menschlichen Mehrwert — YouTubes Sperrziel 2026. |
| **Disclosure / „Altered content"** | KI-Kennzeichnung beim Upload (Pflicht für realistische KI-Inhalte). |
| **C2PA** | *Coalition for Content Provenance and Authenticity* — Metadaten-Standard, der Herkunft/KI-Erzeugung im File markiert; schützt vor falschem Auto-Label. |
| **Content ID** | YouTubes Rechte-Erkennung; Musik muss „Content-ID-sicher" lizenziert sein (Artlist/Epidemic), sonst Claims. |
| **Image→Video** | KI-Workflow: erst Standbild generieren, dann animieren (Higgsfield Soul → Kling/Veo). |
| **TTS** | *Text-to-Speech* — KI-Sprachausgabe (Higgsfield Audio, ElevenLabs). |
| **n8n** | Open-Source-Automatisierungstool (Workflows); hier später für Mechanik (Upload/Scheduling). |
| **Quota (API)** | Tageskontingent der YouTube-API; `videos.insert` = 1.600 Units, Standard 10.000/Tag (~6 Uploads). |
| **publishAt** | API-Feld für terminierte Veröffentlichung (ISO-8601-UTC), erzwingt vorher „private". |
